12. 학술 영어 : 최신 기술을 획득하기 위한 영어 문서나 논문 공부

19. 자연어 처리 : RNN, LSTM, Seq2Seq를 넘어서 최근 자연어 처리의 핵심이라고 할 수 있는 **ansformer BERT**GPT
